This project was generated with Angular CLI version 1.3.0. It is a Front-end Angular 4 application and it requires a backend application.
Prerequisite
  • Java JDK 8
  • Git
Clone the server side application
$ git clone https://github.com/ivans-innovation-lab/my-company-monolithic-web.git
Run the server side application
$ cd my-company-monolithic-web
$ ./mvnw clean install
$ ./mvnw spring-boot:run
